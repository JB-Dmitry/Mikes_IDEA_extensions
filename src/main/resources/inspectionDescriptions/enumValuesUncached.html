Reports use of values() method on Enum types.

values() is internally implemented as return $VALUES.clone();. It's recommended co call values() only once and use this array instance further. Sample: 

    private static final ThisEnum[] VALS = values();

    /**
     * Number of available enum constants.
     * @see #valueAt
     */
    public static final int THIS_ENUM_COUNT = VALS.length;

    /**
     * Returns enum constant at the given index.
     * @param index must be in [0; THIS_ENUM_COUNT)
     * @return enum constant at the given index
     * @throws ArrayIndexOutOfBoundsException for wrong indices
     * @see #THIS_ENUM_COUNT
     */
    public static ThisEnum valueAt(int index) {
        return VALS[index];
    }
